# APMeter
Tool for game Aion (RU). Count Abyss Points from logs.

Для корректной работы необходимо:

1. Включить логироввание чата.
Для этого нужно добавить файл system.ovr в корень игры Aion.
Файл должен содержать следующие строчки
g_save_preset = "1"
g_chatlog = "1"
log_Verbosity = "1"
log_FileVerbosity = "1"
log_IncludeTime = "1"

2. При работе программы должна быть включена РУ расскладка клавиатуры. 
Иначе в лог файле будут крякозябры вместо символов по которым программа не может подсчитать что-либо.

Не забывайте время от времени чистить лог.
